Builds on Semaphore run on Ubuntu 12.04 Precise LTS. The following Ruby versions are available via "rbenv":https://github.com/sstephenson/rbenv/:

* 2.0.0-p0
* 1.9.3-p374 (p327 is also available)
* 1.9.2-p320
* ree-1.8.7-2012.02
* JRuby 1.7.1
* Bundler 1.2.4 (on all)
