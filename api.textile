Semaphore has an API for viewing the status of project's branch.

To see how to use the API for your project, follow these steps:

# Click on "settings" next to your project on the dashboard.
# Open the "API" tab on project settings page.
# Select a branch of which you want to view the status.
# Perform a HTTP GET request on the provided URL.
