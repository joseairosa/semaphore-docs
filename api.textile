Semaphore provides an API for the following:

* listing of user's projects
* listing of project's branches
* status of a project's branch
* history of a project's branch

p(accent). All API access is over HTTPS. Every method requires the user to provide his authentication token via <strong>auth_token</strong> parameter.

Example request:

<pre>$ curl -i https://semaphoreapp.com/api/v1/projects?auth_token=Yds3w6o26FLfJTnVK2y9</pre>

To see your authentication token and request URIs of a specific project, open a project's settings, then find the "API" tab.

h3. 4. Branch History

<pre>GET /api/v1/projects/:hash_id/:id</pre>

h4. Arguments

# <code>hash_id</code> of the project
# <code>id</code> of the branch

Branch builds are returned paginated by 10 per page. A specific page can be requested using the <code>page</code> parameter. The pagination information is included in the response header.
